# GroovyMC Example Modules

Welcome to the **GroovyMC Examples** directory!

Here you will find a collection of ready-to-use modules designed to demonstrate the power and flexibility of the **GroovyMC ScriptAPI**. Whether you are a beginner looking to write your first command or an advanced user integrating databases and custom GUIs, these examples serve as practical learning resources.

---

## Recommended Starting Point

### [example_module](./example_module)
If you are new to GroovyMC, **start here!**

This is the comprehensive "All-in-One" showcase module. It is designed to cover almost every aspect of the API in a single, modular project. It includes:
* **Lifecycle Management:** `onEnable`, `onDisable`
* **Commands:** Arguments, permissions, and Tab Completion
* **Database:** Async SQL queries with Fluent API
* **Configuration:** Custom config files and serialization
* **GUIs:** Inventory menus and click handling
* **Visuals:** Scoreboards (Sidebar), BossBars, and Titles
* **Events:** Listeners for server events

## How to Use

1.  **Download** the folder of the module you want to use (e.g., `example_module`).
2.  **Move** the folder into your server's plugins directory:
    `plugins/GroovyMC/modules/`
3.  **Reload** GroovyMC using the command:
    `/groovy reload`
4.  **Check the console** to see the module load successfully!
